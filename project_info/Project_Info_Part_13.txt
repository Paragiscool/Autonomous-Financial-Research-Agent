SECTION E4: DATA SOURCES AND APIS 
E4.1 SEC EDGAR (Free, No API Key Required) 
The SEC's EDGAR full-text search system (efts.sec.gov) provides free access to all public company 
filings. The API supports full-text search across filings, retrieval by company ticker or CIK number, 
and filtering by filing type and date range. Rate limit: 10 requests per second with a mandatory 
User-Agent header identifying your application. 
 
E4.2 Financial Data APIs 
●​ Alpha Vantage: Free tier provides 25 requests per day. Covers US stocks, fundamentals, 
earnings, and economic indicators. 
●​ Financial Modeling Prep (FMP): Free tier provides 250 requests per day. Covers financial 
statements, ratios, earnings transcripts, and company profiles. 
●​ yfinance (Python library): Free, unofficial Yahoo Finance API wrapper. No rate limit 
documentation but aggressive scraping may result in IP blocking. 
 
E4.3 News And Search APIs 
●​ Tavily: AI-optimized search API designed for agent use. Free tier: 1,000 searches per month. 
Returns clean, structured results ideal for LLM consumption. 
●​ SerpAPI: Google Search results API. Free tier: 100 searches per month. 
●​ NewsAPI.org: Free tier: 100 requests per day. Good for recent news but limited to articles 
from the past month on free tier.
